denoiser — OIDN denoising in the browser, full WebGPU

Intel Open Image Denoise (OIDN) U-Nets, running entirely on WebGPU via onnxruntime-web. No native, no server round-trip — the noisy frame goes in, the clean frame comes out, on the GPU. These demos need a WebGPU browser (Chrome/Edge 113+, Safari 26+).
