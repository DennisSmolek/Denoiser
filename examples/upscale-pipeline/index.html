denoiser · upscale pipeline

Three pmndrs packages on one GPUDevice: three.js renders a noisy scene at low resolution, denoiser cleans it, and @pmndrs/upscaler (AMD FSR) upscales it to full canvas resolution — no CPU pixel copies in the chain. Each GPU texture is handed straight to the next stage on the device the denoiser created. Upscaling here uses FSR1 (spatial): the denoiser is a discrete, stills-oriented pass, which composes cleanly with FSR's per-frame stateless spatial path (the temporal path wants per-frame motion vectors — see the note below).
